# GeradorFormulario
> Escreva pouco e crie formularios com base de templates criado por você

[demo](https://viniceosm.github.io/GeradorFormulario/)

## Template atual
Editor:
``
NomeInput.idTxt
``

Resultado:
```
NomeInput<br>
<input type="text" id="idTxt" placeholder="NomeInput">
```
